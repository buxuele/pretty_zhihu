### 使用方法

1. 打开 Chrome 浏览器， 输入 `chrome://extensions/` 进入扩展程序页面。
2. 打开右上角的开发者模式， 点击 `加载已解压的扩展程序` 按钮，  
3. 把此项目整个文件夹，直接拖入到扩展程序页面
4. 打开知乎专栏页面， 刷新页面， 即可看到效果。
